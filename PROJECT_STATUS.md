## 状態

手書きキャンバスMVPをmainへマージし、GitHub Pagesで公開済み。

## 目的

白いキャンバスに1日分の勉強計画を自由に手書きし、後からタスクとして管理できるアプリを目指す。最重要価値は、今日何をするか決めること。

## 技術

- HTML / CSS / JavaScript
- Canvas API / Pointer Events
- localStorage
- GitHub Actions
- GitHub Pages
- 外部ライブラリ、バックエンド、APIキーなし

## 保存データ

日別ページ: `study-canvas:pages:v2`

タスクカード: `study-canvas:tasks:v1`

週間目標: `study-canvas:weekly:v1`

自由ノート: `study-canvas:notes:v1`

復元前安全コピー: `study-canvas:restore-safety:v1`

移行前データのキー: `study-canvas:drawing:v1`（安全のため削除せず保持）

同じSafari内で日付ごとに1ページを保存する。Safariのサイトデータを削除すると消える可能性があり、別端末には同期されない。

メニューの「バックアップを保存」から、日別・タスク・週間・自由ノートを`study-canvas-backup-YYYY-MM-DD.json`として端末へ書き出せる。v1バックアップの日別手書きも読み込める。復元時は現在の全データを安全コピーとして残す。
